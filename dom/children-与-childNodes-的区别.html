children 与 childNodes 的区别

2017年4月18日 DOM

Node.childNodes

返回包含指定节点的子节点的集合，包含所有节点 (包括元素节点 element，属性节点 attr，文本节点 text 等等)

ParentNode.children

返回一个包含当前元素的子元素的集合 (只包含元素节点 element)

<div>测试</div>

var a = document.querySelector('div');
a.children.length;  // 0
a.childNodes.length;  // 1
